What is Betty - Beauty Salon and Makeup Template

" Betty - Beauty Salon and Makeup Template " is perfect if you like a minimal, clean and modern design. This template is ideal for beauty salons, makeup studio, hairdresser, nail salon, skin care shop, cosmetic health, spa salon, lifestyle and those who need an easy, attractive and effective way to share their work with clients.


We used Latest Bootstrap 5, Flexslider, Owl Carousel, Animate, Themify Icons, Magnific Popup Gallery and Google Fonts in this template. Links are available in Sources and Credits field.


Installation and customization about

Betty template is a static HTML template. Not WordPress. If you want to publish / install as HTML, you can edit and publish with any editor (brackets, notepad++ etc). For this, you need to have basic knowledge of HTML and CSS. When you open HTML extensions in any editor, you can view and edit the description titles I mentioned. (Add your images and content articles in the relevant parts of the template.) Then connect to FTP and send your files to the server. Your site will work. There is no need for any installation. Or if you want to make a custom panel for this template, you can easily customize it.


Notice

All images in Betty template can be used only on demo version. You will need to upload your own images for your site.
